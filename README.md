# **Windows Helper Toolkit**

**Prosty, terminalowy panel administracyjny dla Windows w Pythonie.**  
Pozwala w łatwy sposób sprawdzić informacje o systemie, listę zainstalowanych programów i wyczyścić foldery tymczasowe.

---

## **Funkcje**

- **Wyświetlanie informacji o systemie:** wersja, architektura, użytkownik  
- **Lista wszystkich zainstalowanych programów** (klasycznych)  
- **Czyszczenie folderów tymczasowych Windows** (`Temp` i systemowych)  
- **Interaktywny panel w terminalu** z prostym menu  

---

## **Jak używać**

1. Pobierz plik **`windows_helper.py`**  
2. Zainstaluj Python: [https://www.python.org](https://www.python.org)  
3. Otwórz terminal w folderze z plikiem  
4. Wpisz:
python windows_helper.py

5. Korzystaj z menu, wybierając opcje numeryczne  

---

## **Bezpieczeństwo**

- Skrypt używa **tylko standardowych funkcji Windows i Pythona**  
- Nie modyfikuje rejestru ani plików systemowych poza folderami tymczasowymi
